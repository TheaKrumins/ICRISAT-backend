In this folder are located the Excel templates to be used by "Import" option of the dietary system. There are five templates:

    ConvFactors.xls            : Conversion factors
    Fct.xls                    : Food composition
    Groups.xls                 : Groups
    Recipes.xls                : Recipes
    RetentionFactors.xls       : Retention factors

In the folder ImportExamples there is some little examples of the templates.

WARNING: The name of each file, the column names and the sheet names MUST NOT BE changed!!
